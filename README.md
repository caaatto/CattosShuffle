# CattosShuffle

A fun and thrilling equipment & bag roulette addon for World of Warcraft Classic Era (Hardcore).

## Features

- **Equipment Roulette**: Randomly highlights equipment slots with various animation styles
- **Bag Roulette**: Spin to select random bags for emptying or deletion
- **Multiple Animation Styles**: Including normal, reverse, fakeout, yoyo, kitt, and more
- **Custom Sound Effects**: Immersive casino-style sounds during spins
- **Minimap Button**: Quick access to the addon interface
- **Localization**: Supports English and German

## Installation

1. Download the latest release
2. Extract the `CattosShuffle` folder to your WoW AddOns directory:
   - Windows: `World of Warcraft\_classic_era_\Interface\AddOns\`
   - Mac: `World of Warcraft/_classic_era_/Interface/AddOns/`
3. Restart WoW or type `/reload` in chat

## Usage

### Commands
- `/cattos` or `/cc` - Open the main interface
- `/cattos bags` - Start bag roulette
- `/cattos sheet` - Start equipment roulette
- `/cattos delete` - Start bag deletion roulette
- `/cattos choice` - Free choice mode
- `/cattos sound [theme]` - Change sound theme (trute/default)

### Interface
- Click the minimap button to open the interface
- Select which slots to include in the roulette
- Choose your action and spin!

## Sound Themes

- **Trute**: Custom casino-style sound effects
- **Default**: Standard WoW interface sounds

## Configuration

Access settings through:
- ESC → Interface → AddOns → CattosShuffle
- Or use the Options button in the addon interface
